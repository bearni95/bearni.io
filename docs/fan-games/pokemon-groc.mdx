# Pokemon Groc

> Traducció del disassembly de Pokemon Yellow 

<img class="project-icon" src="/img/projects/pokemon-groc.jpg"/>



## Descàrrega
Pots baixar l'última versió des de Github, en [aquest enllaç](https://github.com/bearni95/pokemon-groc/releases), triant `pokemon-groc.gb`

## Com jugar-hi
El joc està compilat com un de Game Boy: un cop descarreguis `pokemon-groc.db` des de Github, pots jugar com si fos un ROM corrent de Pokémon, en qualsevol emulador de GameBoy i GameBoy Color
