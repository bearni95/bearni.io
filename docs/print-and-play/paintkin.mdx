# PaintKin

> Print, paint and play! The first DIY card game for all ages

![cover](/img/projects/paintkin/cover.png)

## The Cards

The game's cards are 63x88mm, the same size as the Magic The Gathering, and Pokemon Cards. This is mostly to allow
using standard sleeves for those games to preserve and improve the print-at-home cards.

All cards in the base set can be downloaded, either in full color, or in printer-friendly, so to color them after printing.

## The Game

![rules](/img/projects/paintkin/rules.png)

Each of the PDFs with the card decks includes the instructions for the **One Up** game, a simple yet challenging 1v1
card game for ages 5 and above.
